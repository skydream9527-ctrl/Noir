# 极简浏览器设计规格书

**项目**：极简浏览器 Android 应用  
**版本**：v2.0  
**日期**：2026-04-10  
**状态**：已批准

---

## 1. 设计理念

打造一款"功能完整、视觉简洁"的浏览器。保留标签页、搜索引擎、书签、历史等核心功能，同时通过卡片式设计、大圆角、大间距、线性图标等元素，营造简洁、呼吸感的视觉体验。

---

## 2. 视觉风格

### 色彩系统

| 用途 | 颜色 | 色值 |
|------|------|------|
| 背景 | 纯白 | `#FFFFFF` |
| 卡片/输入框 | 浅灰白 | `#F8F9FA` |
| 强调色（激活态） | Google蓝 | `#1A73E8` |
| 文字主色 | 深灰黑 | `#202124` |
| 文字次色 | 中灰 | `#5F6368` |
| 分割线 | 浅灰线 | `#E8EAED` |

### 圆角与间距

| 元素 | 圆角 | 间距 |
|------|------|------|
| 输入框/卡片 | 16dp | 24dp（模块间） |
| 按钮/列表项 | 12dp | 16dp（卡片内） |

### 图标风格

- 线性图标（stroke-only）
- 尺寸：24dp
- 线宽：2dp
- 未激活：`#5F6368`
- 激活：`#1A73E8`

### 字体

- 系统默认字体
- 标题：16sp medium
- 正文：14sp regular
- 辅助：12sp regular

## 4. 组件设计

### 4.1 地址输入框

**状态1 - 默认**
```
┌────────────────────────────────┐
│  🔍 输入网址或搜索               │  ← 灰色placeholder
└────────────────────────────────┘
```

**状态2 - 输入中**
```
┌────────────────────────────────┐
│  🔍 github.com                  │
│                           [×]  │  ← 清除按钮
└────────────────────────────────┘
```

**状态3 - 加载中**
```
┌────────────────────────────────┐
│  🔍 github.com                  │
│  ▓▓▓▓▓▓▓▓░░░░░░░░░░  40%       │  ← 进度条
└────────────────────────────────┘
```

### 4.2 抽屉交互

- **呼出**：右滑手势 或 点击地址栏左侧菜单图标
- **关闭**：左滑 或 点击遮罩层
- **Tab切换**：点击切换书签/历史/标签页

### 4.3 标签页卡片

- **点击**：切换到该标签页
- **点击 [×]**：关闭标签页
- **长按**：显示关闭/收藏等选项菜单

### 4.4 手势优先级

1. 下滑 → 显示隐藏的工具栏
2. 右滑 → 呼出抽屉
3. 点击地址栏 → 聚焦输入

## 5. 架构设计

### 5.1 模块划分

```
┌─────────────────────────────────────┐
│              UI层                   │
│  MainActivity / BrowserActivity     │
├─────────────────────────────────────┤
│             业务层                   │
│  TabManager / BookmarkManager      │
│  HistoryManager / SearchEngineMgr   │
├─────────────────────────────────────┤
│             数据层                  │
│  SharedPreferences (本地存储)        │
│  WebViewCache                       │
├─────────────────────────────────────┤
│             核心层                  │
│  X5 WebView (腾讯X5内核)            │
│  SystemWebView                      │
└─────────────────────────────────────┘
```

### 5.2 关键类职责

| 类名 | 职责 |
|------|------|
| `BrowserActivity` | 主浏览页面，承载WebView，处理地址栏交互 |
| `TabManager` | 标签页管理（创建/切换/关闭/持久化） |
| `BookmarkManager` | 书签CRUD操作 |
| `HistoryManager` | 历史记录管理 |
| `SearchEngineManager` | 搜索引擎管理（添加/切换/默认） |
| `DrawerController` | 右侧抽屉呼出/关闭/遮罩控制 |

### 5.3 数据流

```
用户操作 → Activity → Manager → SharedPreferences
                                    ↓
WebView事件 ← Activity ← Manager ←────
```

## 7. 实现优先级

### 第一阶段：核心浏览
1. 重构 BrowserActivity 布局（底部地址栏）
2. 实现合并地址栏（搜索/URL自动判断）
3. 集成 X5 WebView

### 第二阶段：标签系统
4. TabManager 重构
5. 标签页切换 UI
6. 标签页持久化

### 第三阶段：抽屉功能
7. DrawerController 实现
8. 书签管理（BookMarkManager）
9. 历史记录（HistoryManager）

### 第四阶段：优化
10. 搜索引擎选择器
11. 动画和过渡效果
12. 深色模式适配
